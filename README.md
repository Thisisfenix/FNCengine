# Friday Night Cooking Engine 🍳

Un motor de juegos culinarios con múltiples modos de juego, desarrollado en HTML5, CSS3 y JavaScript puro.

## 🚀 Características principales

- **Tres juegos en uno**:
  - 🎮 Pepe's RPG Adventure (Aventura RPG)
  - 🍗 Pollolito Challenge (Contra reloj)
  - 🧠 Memorama Culinario (Juego de memoria)
  
- **Sistema personalizable**:
  - 🎨 Selector de colores para la interfaz
  - 🌙 Modo claro/oscuro automático
  - 🔊 Control de volumen independiente para música y efectos

- **Logros y secretos**:
  - 🏆 12+ logros desbloqueables
  - 🔓 Modo secreto con código Konami (↑↑↓↓←→←→BA)

## 🎮 Cómo jugar

| Juego | Controles | Descripción |
|-------|-----------|-------------|
| **Pepe's RPG** | Teclado + Ratón | Aventura por turnos |
| **Pollolito** | Ratón/Táctil | Cocina contra reloj |
| **Memorama** | Clics/Touch | Encuentra los pares |
